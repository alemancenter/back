# Chatbot Knowledge Base Expansion

تم توسيع قاعدة معرفة مساعد المنصة لتغطية أكثر الأسئلة والمشاكل المتوقعة من الزوار والأعضاء داخل Alemancenter.

## المجالات المغطاة

- تسجيل الدخول وكلمة المرور.
- إنشاء الحساب والبريد المستخدم سابقًا.
- تفعيل البريد ورسائل التحقق التي لا تصل.
- تحميل الملفات وروابط التحميل المؤقتة والصلاحيات.
- الملفات غير الموجودة أو الروابط القديمة.
- البحث عن الملفات حسب الصف والمادة والفصل.
- الدولة والمنهج وتصنيف المحتوى.
- مشاكل الموقع العامة مثل 404/500/502 والبطء ومشاكل الهاتف.
- تعديل بيانات الحساب والخصوصية.
- التواصل مع الإدارة والإبلاغ عن ملفات خاطئة وطلب إضافة محتوى.

## طريقة العمل

يتم إدخال هذه المعرفة تلقائيًا من خلال `seedDefaultKnowledge` عند تهيئة جداول البوت. إذا كانت الأسئلة موجودة سابقًا، يتم تحديث الإجابات والكلمات المفتاحية والأولوية بدل تكرار السجلات.

## ملاحظات تشغيل

- لا يحتاج الأمر إلى تنفيذ SQL يدويًا في الوضع الطبيعي.
- يجب إعادة بناء الباك اند وتشغيله حتى يتم تطبيق التحديثات.
- عند وجود سجلات قديمة في جدول `chat_knowledge_base` بنفس السؤال و`country_code = all` سيتم تحديثها تلقائيًا.

## ملفات تم تعديلها

- `internal/repositories/chatbot/repository.go`
- `internal/services/chatbot/service.go`
