# Chatbot Feature Guard — Email Only

تمت إضافة حارس ميزات للبوت لمنع ذكر خصائص غير موجودة في المنصة حاليًا.

## ما تم منعه

- إنشاء حساب عبر رقم الهاتف.
- تفعيل الحساب عبر رقم الهاتف.
- تفعيل SMS.
- تفعيل WhatsApp.
- تسجيل الدخول باسم المستخدم.
- استعادة كلمة المرور عبر الهاتف أو WhatsApp.

## السلوك الجديد

إذا سأل المستخدم مثلًا:

- بزبط بدون بريد؟
- هل يمكن التسجيل برقم الهاتف؟
- هل يصلني الكود على واتساب؟
- أقدر أدخل باسم المستخدم؟

يرجع البوت جوابًا واضحًا:

> حاليًا لا تدعم المنصة إنشاء الحساب أو تفعيل الحساب أو تسجيل الدخول عبر رقم الهاتف أو WhatsApp أو SMS أو اسم المستخدم. الطريقة المتاحة الآن هي البريد الإلكتروني فقط.

## الحماية

- حارس مبكر قبل AI وقبل قاعدة المعرفة.
- فلتر بعد الرد يمنع خروج إجابات من AI أو knowledge_base تتحدث عن هاتف/واتساب/SMS كميزة تسجيل أو تفعيل.
- Prompt الذكاء الاصطناعي تم تحديثه بقواعد صارمة.
- لم يتم منع عبارات الهاتف الخاصة بالجهاز مثل: تحميل من الهاتف أو الموقع لا يعمل على الهاتف.
