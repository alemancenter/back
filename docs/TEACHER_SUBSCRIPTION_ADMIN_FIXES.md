# إصلاحات إدارة اشتراكات المعلمين

## تم إصلاح
- خطأ Next.js الخاص بـ params.section في dynamic route عبر useParams.
- إضافة قسم علوي مستقل في Sidebar باسم: اشتراكات المعلمين.
- إضافة aliases لصلاحية manage teacher subscriptions حتى تظهر للقائمين بالإدارة عند وجود manage users/roles/settings.
- إضافة إيقاف اشتراك من صفحة الاشتراكات.
- إضافة حذف عضوية المعلم من Teacher Pro من صفحة المعلمين المشتركين.

## الفرق بين حذف العضوية وحذف المستخدم
زر حذف العضوية لا يحذف حساب المستخدم من جدول users.
يقوم فقط بـ:
- إلغاء الاشتراكات النشطة
- إزالة دور Teacher Pro
- تعطيل أجهزة المعلم
- حذف teacher_profile
- مسح كاش المستخدم
