# Teacher Membership Revoke Hard Fix

تم إصلاح مشكلة بقاء المعلم قادرًا على استخدام خدمات Teacher Pro بعد إيقاف أو حذف العضوية.

## الإصلاحات
- إيقاف الاشتراك لم يعد يعتمد فقط على status.
- عند إيقاف آخر اشتراك نشط:
  - يتم إزالة دور Teacher Pro.
  - يتم حذف صلاحيات المعلم المباشرة من model_has_permissions.
  - يتم تعطيل الأجهزة.
  - يتم مسح كاش المستخدم من Redis.
- عند حذف عضوية المعلم:
  - يتم إلغاء كل الاشتراكات النشطة.
  - إزالة Teacher Pro.
  - تعطيل كل الأجهزة.
  - حذف teacher_profile.
  - إزالة أي صلاحيات مباشرة للمعلم.
  - مسح كاش المستخدم.
- تم إضافة فحص فعال داخل handlers الخاصة بخدمات المعلم:
  - workspace
  - files
  - library
  - downloads
  - ai-generations
  - devices
  بحيث لا يكفي بقاء الدور في الواجهة؛ يجب وجود اشتراك active في قاعدة البيانات.
- تم إضافة Cleanup تلقائي عند تشغيل الباك إند:
  أي مستخدم يحمل Teacher Pro ولا يملك اشتراكًا نشطًا يتم تنظيفه تلقائيًا.

## ملاحظة
بعد رفع الإصلاح يجب تنفيذ redis-cli FLUSHDB وإعادة تسجيل دخول المستخدم المتأثر أو تحديث جلسته.
