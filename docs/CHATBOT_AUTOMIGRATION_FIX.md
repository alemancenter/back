# Chatbot Auto Migration Fix

تم تعديل نظام Chatbot حتى لا يحتاج تنفيذ SQL يدويًا في التشغيل الطبيعي.

## ما تم إصلاحه

1. إضافة AutoMigrate احتياطي داخل مستودع البوت عند أول استخدام لكل قاعدة دولة.
2. إصلاح قيمة `chat_messages.metadata` حتى تكون دائمًا JSON صالحًا `{}` بدل النص الفارغ.
3. إضافة Seed تلقائي لأول أسئلة معرفة أساسية: تسجيل الدخول، التحميل، وتفعيل البريد.
4. إبقاء AutoMigrate الأساسي في `cmd/server/main.go` كما هو عند تشغيل السيرفر.

## ملاحظة مهمة

`go build` لا ينشئ الجداول. الجداول تنشأ عند تشغيل الباك اند لأن الكود يتصل بقاعدة البيانات وقت التشغيل وليس وقت البناء.

إذا كانت صلاحيات مستخدم قاعدة البيانات تمنع CREATE/ALTER TABLE، يجب إعطاء الصلاحيات أو تنفيذ SQL يدويًا مرة واحدة.
