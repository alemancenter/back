# Chatbot Database Content Search Engine

تم تطوير بحث بوت الدردشة ليقرأ مباشرة من قاعدة البيانات داخل:

- files
- articles
- posts

## التحسينات
- البحث يبدأ بالملفات أولًا لأن أغلب الزوار يبحثون عن اختبارات/نماذج/ملفات.
- البحث في المقالات والمنشورات بعد الملفات.
- دعم مرادفات عربية:
  - امتحان / اختبار / نموذج
  - نهائي / نهاية
  - نموذج2 / نموذج 2
  - فيزياء / فيزيا / فزيا
  - عربي / اللغة العربية
  - انجليزي / انكليزي
- إرجاع معلومات إضافية للبوت: الصف، المادة، الفصل، التصنيف.
- ترتيب النتائج حسب درجة تطابق Score.
- منع الاعتماد على HTML فقط أو صفحات عامة.
- الروابط المعروضة تأتي من قاعدة البيانات فقط.
