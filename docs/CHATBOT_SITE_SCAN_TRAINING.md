# Chatbot Site Scan Training

تم تطوير تدريب بوت الدردشة بناءً على قراءة صفحات الموقع الحية:
- الصفحة الرئيسية
- صفحة الأردن /jo
- صفحة الصف التاسع
- صفحة رياض الأطفال
- صفحة المقالات /jo/posts
- صفحة الخدمات /services
- صفحة من نحن /about-us
- صفحة اتصل بنا /contact-us
- صفحة البحث /search
- صفحة تسجيل الدخول /login

## أهم التعديلات
- توجيه عرض الصفوف إلى /jo بدل /classes لأن صفحة الأردن تعرض الصفوف فعليًا.
- تحسين رد "كيف أستخدم الموقع؟" بناءً على فلاتر الموقع: الصف، نوع المحتوى، المادة، الفصل.
- تحسين رد "خدمات الموقع" حسب صفحة الخدمات.
- تحسين رد "من نحن" حسب صفحة من نحن.
- تحسين رد التواصل: استخدام /contact-us، وإذا ظهر خطأ reCAPTCHA أو نموذج غير مهيأ يُذكر البريد info@alemancenter.com.
- منع تحويل أسئلة الصفوف والخدمات ومن نحن إلى بحث محتوى أو AI.
- تحديث روابط الإجراءات في buildActions.
